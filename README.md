# Heart Notes Landing Page

A simple, elegant, mobile-first landing page for **Heart Notes**, a premium car perfume brand.

## Live Preview

https://sa-od.github.io/heat-note/

## Features

- Mobile-first responsive layout
- Tailwind CSS styling
- Luxury monochrome brand aesthetic
- Hero banner, perfume collection, and combo sections
- Clean single-page structure

## Products

- Nitro Black
- Carbon White
- Special Edition
- 4 combo editions

## Tech

- HTML
- Tailwind CSS

## Usage

1. Open the project files
2. Replace the banner image path with your own image
3. Update product text or links if needed
4. Deploy to GitHub Pages or any static hosting
